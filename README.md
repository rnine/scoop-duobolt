# Scoop Bucket for duobolt

Fast duplicate file finder using BLAKE3 hashing.

## Installation

```powershell
scoop bucket add duobolt https://github.com/rnine/scoop-duobolt
scoop install duobolt
```

## Usage

```powershell
duobolt-cli --help
duobolt-cli scan C:\path\to\folder
```
